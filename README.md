Student Management Dashboard – Power BI Project

Project Overview

This project is a Power BI dashboard created for analyzing student personal details and course-related information. The dashboard helps track student data, joining trends, certificates issued, and course analytics in an interactive way.

---

Tools & Technologies Used

- Power BI
- MySQL
- SQL Queries
- Data Visualization
- Power Query
- DAX Basics

---

Dataset Information

The project contains:

- Student Personal Details
- Student Course Details

Sample Fields

- Student Name
- Age
- Contact Number
- Email ID
- City
- State
- Course Name
- Joining Year
- Issued Certificate

SQL Operations Performed

- CREATE DATABASE
- CREATE TABLE
- INSERT INTO
- SELECT Queries


Dashboard Features

- Total Students KPI
- Course-wise Analysis
- Certificate Status
- City-wise Student Distribution
- Joining Year Analysis
- Interactive cards,gauge,Filters & Slicers


Database Connection

The dashboard is connected with MySQL database using:

- MySQL Workbench
- MySQL Connector/NET
- Power BI MySQL Connector
